This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

It's aimed to provide a simple parcel order workflow: create a delivery request in a 3-step form, validate input data, and store orders in browser local storage.
The app also includes order history with search/filter, order details pages, and delete confirmation modal.
UI is built with Next.js App Router and Tailwind CSS with responsive layouts for mobile and desktop.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
